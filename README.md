A modern, responsive online bookstore web application built with HTML, CSS, JavaScript, and Firebase. Users can browse books, add them to a shopping cart, manage their account, subscribe to newsletters, and checkout their orders.

---

##  Live Demo

Access the live site hosted on Firebase: [https://bookstore-ffcd9.web.app/](https://bookstore-ffcd9.web.app/)

---

## Features 

- **Responsive Design**: Works on desktops, tablets, and mobile devices.
- **Browse & Search Books**: Explore books by genre, author, or title.
- **Shopping Cart**:
  - Add books to cart
  - Update quantities
  - Remove items
  - View total price
- **User Authentication**:
  - Login and Register using Firebase Authentication
  - Persistent sessions
- **Firebase Firestore**:
  - Store users, cart data, and newsletter subscriptions
- **Newsletter Subscription**: Users can subscribe to updates.
- **Toast Notifications**: Informative alerts when actions are performed.
- **Checkout Process**: Saves orders to Firebase Firestore.
